# Simulador estúpido de sistema masa-resorte de 2 dimensiones
Este es un simulador simple que hace uso de las leyes de Newton para calcular los movimientos
de cada uno de los nodos de nuestra tela y usa *openGL* para su dibujado en pantalla. 

Este proyecto es para fines didácticos, por lo que muchas de las técnicas empleadas no son 
las más eficientes para el dibujado o cálculo de las posiciones. Es por esta misma razón
(el que este proyecto sea para yo aprender) también que trato de definir yo mismo los tipos
de datos y métodos necesarios para desarrollar el proyecto. Quien quiera que esté viendo esto, 
mírenlo como si fuese un _hola mundo_ en mis habilidades con C++ y en github.
